QuickMafs

Javascript NodeJS HTML CSS

QuickMafs est une application Web de calcul mental multijoueur. Le jeu à été initialement développé pour jouer avec des amis.
Les règles sont simples: cinq nombres sont générés, puis avec des opérations de bases (+, -, x, /), trouver le plus rapidement possible un nombre un plus grand (48 sur l'image).

Lien du jeu: https://quickmafs.herokuapp.com/
Lien Demo: https://www.youtube.com/watch?v=3lBYJQ3lHGE

LocateIt

Javascript PHP MySQL HTML CSS

LocateIt est une application Web où le but est de localiser un pays sur une carte intéractive.
Des points et des indices ont été mis en place, comme à la manière de Geoguessr plus on a de points, meilleur est le score. Le score maximal est de 5000 points.
D'autres fonctionnalité comprises: Authentification, historique, carte customisable, page de solutions et des indices.

Lien du jeu: Pas hébergé
Lien Demo: https://www.youtube.com/watch?v=u1RO4DPsHLE
